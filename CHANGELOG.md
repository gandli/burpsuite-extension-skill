## [1.0.0] - 2026-05-30

- Initial release of Burp Suite Extension Development Skill

- Python development support with Jython (Legacy Extender API)
- Multi-platform environment setup guides (macOS, Windows, Linux)
- Project templates for Java and Python
- Project generator scripts
- Best practices documentation
- Debugging guides
- Common issues and solutions
- Official resource links
- TRACE evaluation framework optimization

### Features

- **Dual Language Support**: Java (Montoya API) and Python (Jython)
- **Project Templates**: Ready-to-use templates for quick start
- **Project Generator**: CLI tool for scaffolding new projects
- **Multi-Platform**: Complete setup guides for macOS, Windows, and Linux
- **Best Practices**: Error handling, performance, logging, and security
- **Examples**: Real-world code examples (Request Logger, Security Header Check)

### Documentation

- Comprehensive SKILL.md with TRACE-optimized structure

- FAQ section for common questions
